# Attendance Tracker

A web application built using Flask that helps students manage and track attendance records.

## Features

- User Registration
- User Login & Authentication
- Personalized Dashboard
- Attendance Tracking
- Attendance Percentage Calculation
- Session Management
- Secure Logout

## Tech Stack

- Python
- Flask
- HTML
- CSS
- Jinja2

## Learning Objectives

This project was built to strengthen my understanding of:

- Backend Web Development
- Flask Routing
- Authentication & Sessions
- Database Integration
- CRUD Operations
- Full-Stack Application Development
